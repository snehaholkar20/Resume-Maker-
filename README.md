# Resume-Maker 📄

A web application to create and download professional resumes with ease.

## 📋 Features

- ✨ **User-Friendly Interface** - Simple and intuitive design
- 📝 **Multiple Resume Templates** - Choose from professional templates
- 💾 **Save & Export** - Download resumes as PDF or DOCX
- 🎨 **Customizable Sections** - Add/edit personal details, experience, education, skills
- 🔧 **Real-Time Preview** - See changes as you type
- 📱 **Responsive Design** - Works on desktop and mobile devices
- 🎯 **Professional Formatting** - ATS-friendly resume formats

## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- No installation required - runs entirely in the browser

### Installation

1. Clone the repository:
```bash
git clone https://github.com/snehaholkar20/Resume-Maker-.git
cd Resume-Maker-
```

2. Open the application:
```bash
# Simply open index.html in your browser
open index.html
```

Or use a local server:
```bash
python -m http.server 8000
# Visit http://localhost:8000
```

## 💻 Technologies Used

- **Frontend**: HTML5, CSS3, JavaScript
- **Storage**: LocalStorage (browser-based)
- **Export**: PDF generation library
- **Responsive Framework**: CSS Grid & Flexbox

## 📝 Usage

1. **Fill Your Details** - Enter your personal information
2. **Add Sections** - Include experience, education, and skills
3. **Choose Template** - Select a professional design
4. **Preview** - Review your resume in real-time
5. **Download** - Export as PDF or DOCX

## 🎯 How It Works

1. Enter your personal and professional information
2. The app stores your data in browser storage
3. Select a template to format your resume
4. Preview the resume in real-time
5. Download the final resume in your preferred format

## 🔒 Privacy & Security

- All data is stored locally in your browser
- No information is sent to external servers
- Your resume data remains completely private
